# ClimateQA

A multiple-choice benchmark for evaluating LLM climate science knowledge, grounded in IPCC AR6 source passages across all three Working Groups.

## What's here

172 curated questions generated by four frontier models (Claude Sonnet 4.6, GPT-4o, Gemini 2.5 Flash, Llama 3.3 70B) from 29 IPCC AR6 sections. The full case study with methodology, curation analysis, and evaluation results is in [`CASE_STUDY.md`](CASE_STUDY.md).

## Structure

```
benchmark.json          # Final 172-question dataset
CASE_STUDY.md           # Full write-up: design, results, limitations
workflow.md             # Generation process documentation
scripts/                # Pipeline: generate → review → evaluate → analyse
  generate_prompts.py
  generate_responses.py
  build_review.py
  review_app.py         # Streamlit curation interface
  collate.py
  evaluate.py
  analyse.py
prompts/                # Prompt templates and per-section model responses
passages/               # Source IPCC AR6 passage text
eval/                   # Evaluation results (results.csv, summary.json)
analysis/               # Charts and report generated by analyse.py
literature/             # Reference papers
```

## Running the pipeline

```bash
pip install anthropic openai google-generativeai groq streamlit pandas matplotlib

# Review generated responses
python scripts/review_app.py

# Build benchmark from accepted questions
python scripts/collate.py

# Run evaluation
python scripts/evaluate.py

# Generate analysis charts and report
python scripts/analyse.py
```

All scripts support `--dry-run` and `--overwrite` flags. Evaluation is resumable — completed pairs are skipped automatically.

## Key findings

- **90.1% / 88.4% / 86.6%** accuracy for Claude, GPT-4o, and Llama on the final benchmark
- Data interpretation questions (11pp harder than standard) validated that the embedded-scenario format tests quantitative reasoning, not just recall
- Claude shows a **+9.1pp self-advantage** on questions it generated; GPT-4o and Llama show the opposite — both findings have implications for benchmark construction methodology
- Claude's acceptance rate during curation (46.9%) was nearly double the other models, driven by stronger distractor quality on hard questions
